# percent

### ━ Syntax

<mark style="color:red;">**`[float result] percent(float value, float percentage)`**</mark>

### ━ Usage

```lua
local result = vEngine.math.percent(500, 0.5) --Retrieves specified percentage of the value
print(result) --Prints result
```
