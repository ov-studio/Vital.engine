# tickProgress

### ━ Syntax

<mark style="color:red;">**`[float progress] tickProgress(float tick, float duration)`**</mark>

### ━ Usage

```lua
local result = vEngine.math.tickProgress(500, 60*1000) --Retrieves tick's progress
print(result) --Prints result
```
