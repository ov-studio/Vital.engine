# vector

### ━ Syntax

<mark style="color:red;">**`[userdata vector] vector(float x = 0, float y = 0, float z = 0, float w = 0)`**</mark>

### ━ Usage

```lua
local cVector = vEngine.math.vector.create.create(10, 10, 10, 10) --Creates our vector
print(cVector) --Prints our vector
```
