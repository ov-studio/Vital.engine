# setVSyncEnabled

### ━ Syntax

<mark style="color:red;">**`[bool status] setVSyncEnabled(bool state)`**</mark>

### ━ Usage

```lua
local result = vEngine.config.setVSyncEnabled(false) --Disables our game's VSync mode
print(result) --Prints result
```
