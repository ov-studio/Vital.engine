# wait

{% hint style="danger" %}
**API available only within a valid thread.**

<mark style="color:red;">**`[bool status] wait(int seconds)`**</mark>

```lua

    print("Hello") --Prints "Hello"
    vEngine.thread.wait(5) --Waits 5 second before printing "There"
```
