# create

### ━ Syntax

<mark style="color:red;">**`[userdata color] create(int red, int green, int blue, int alpha)`**</mark>

### ━ Usage

```lua
local cColor = vEngine.color.create(255, 0, 0, 255) --Creates red color
print(cColor) --Prints our color
```
